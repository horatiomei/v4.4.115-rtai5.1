482 /* Get the architecture specific values describing the floating-point
483  evaluation. The following symbols will get defined:
484 
485  float_t floating-point type at least as wide as `float' used
486  to evaluate `float' expressions
487  double_t floating-point type at least as wide as `double' used
488  to evaluate `double' expressions
489 
490  FLT_EVAL_METHOD
491  Defined to
492  0 if `float_t' is `float' and `double_t' is `double'
493  1 if `float_t' and `double_t' are `double'
494  2 if `float_t' and `double_t' are `long double'
495  else `float_t' and `double_t' are unspecified
496 
497  INFINITY representation of the infinity value of type `float'
498 
499  FP_FAST_FMA
500  FP_FAST_FMAF
501  FP_FAST_FMAL
502  If defined it indicates that the `fma' function
503  generally executes about as fast as a multiply and an add.
504  This macro is defined only iff the `fma' function is
505  implemented directly with a hardware multiply-add instructions.
506 
507  FP_ILOGB0 Expands to a value returned by `ilogb (0.0)'.
508  FP_ILOGBNAN Expands to a value returned by `ilogb (NAN)'.
509 
510  DECIMAL_DIG Number of decimal digits supported by conversion between
511  decimal and all internal floating-point formats.

513 */
